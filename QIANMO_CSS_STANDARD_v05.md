# QianMo CSS Standard v05

## Typeface
`font-family: Arial, "PingFang SC", "Microsoft YaHei", "Noto Sans SC", sans-serif;`

## Layout
- Max width: 1320px
- Desktop side padding: 48px
- Tablet side padding: 32px
- Mobile side padding: 20px
- Standard section vertical spacing: 120px
- Card gap: 20–24px

## Radius
- Image/card radius: 10px
- Pill button radius: 999px

## Color
- Black: #050505
- Ink: #111111
- Dark footer: #2b2b2b
- Paper: #f4f1eb
- Line: #d7d3ca
- Muted text: #707070

## CSS components
- `.btn`, `.btn-white`, `.btn-black`, `.btn-light`, `.btn-ghost-dark`
- `.container`, `.section`, `.section-title`, `.kicker`, `.body-copy`
- `.service-card`, `.project-card`, `.metric-card`, `.info-card`
- `.site-footer`, `.footer-columns`, `.footer-bottom`

## Rule
No green UI color. Use black/white/gray only. Color can appear inside project photography.

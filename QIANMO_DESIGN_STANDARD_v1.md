# QianMo Design Standard v1.0

## Positioning
QianMo / 千幕 is a Light Structure Studio: architectural lighting, project support, and digital presentation.

## Visual Direction
Black / white / warm paper / editorial / architectural / restrained. No green UI color. Project imagery may contain color, but interface components remain monochrome.

## Layout
- Max width: 1320px
- Desktop side padding: 48px
- Tablet side padding: 32px
- Mobile side padding: 20px
- Main section spacing: 124–130px
- Card gap: 18–24px
- Large radius: 24px
- Small radius: 14px

## Typography
- Font stack: Inter, Helvetica Neue, Arial, PingFang SC, Microsoft YaHei, sans-serif
- H1 desktop: 96–138px, tight line-height
- H2 desktop: 56–72px
- Body: 16–22px
- Eyebrow: 12px uppercase, letter-spacing 0.28em

## Colors
- Black: #0a0a0a
- Ink: #151515
- Charcoal: #222222
- Muted: #666666
- Warm paper: #f7f4ed
- Warm grey: #e8e4de
- Line: #dad8d2
- White: #ffffff

## Components
- Header: transparent over hero, warm-white after scroll
- Button: black/white pill only
- Cards: editorial mosaic, no green accents
- Footer: dark editorial multi-column footer
